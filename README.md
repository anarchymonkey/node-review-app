# node-review-app
A simple app to post reviews about things (for everyone who wants to learn node)

## Project Structure

* Home Page (/)
* Review Page (/review)
* Add Review (/review/add)
* Modify Review (/review/edit/:id)
* Delete Review (/review/delete/:id)
